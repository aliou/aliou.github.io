Some recent selected projects.

Ace

Ace is a Jekyll Theme. It aims to be as simple as possible, so that it can be customized and build upon.
You can see a demo here, and download the latest version on GitHub.

sql-heredoc.vim

sql-heredoc.vim is a Vim syntax highlighting plugin for SQL heredocs.
You can see installation instructions and download the latest version on GitHub.

Moriarty.vim

Moriarty.vim is a custom Vim theme inspired by Made of Code, Bad Wolf and the Base16 suite of theme.
You can see screenshots and download the latest version on GitHub.

Node Reporter

Node Reporter is a Node module to retrieve and manipulate report files from the Reporter application.
It allows you to filter the reports by date, type of report and the network connection of the phone during the report. You can see download the latest version on npm and see the source on GitHub.
